# traphalo.github.io

A simple personal website built with Jekyll and GitHub Pages.

## About

This is a personal website using Jekyll with the minima theme, hosted on GitHub Pages.

## Structure

- `index.md` - Homepage
- `about.md` - About page
- `blog.md` - Blog listing page
- `_posts/` - Blog posts directory
- `_config.yml` - Jekyll configuration

## Local Development

To run this site locally:

```bash
bundle install
bundle exec jekyll serve
```

Then visit `http://localhost:4000` in your browser.

## Deployment

This site is configured for automatic deployment to GitHub Pages. Simply push to the main branch and GitHub Pages will build and deploy the site automatically.
